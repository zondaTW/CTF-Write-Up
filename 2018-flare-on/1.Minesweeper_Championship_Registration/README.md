## Solution
將壓縮檔解開後，會拿到一個`MinesweeperChampionshipRegistration.jar`的檔案  
看到是`.jar`檔時，就直接開7zip解開後，會有
```
-- META-INF -- MANIFEST.MF
|
|- InviteValidator.class
```
用記事本打開InviteValidator.class就看到flag了。  

Flag: `GoldenTicket2018@flare-on.com`
